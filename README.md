# Simple test case of integration test with Elasticsearch

The code in this repo is intentionally simple. Its sole purpose is to demonstrate that now, with Testcontainers 2.x and Elasticsearch Java 9.x, creating integration tests with Elasticsearch is easier than ever before.

For the full walkthrough, please follow the instructions in the [blog post TBD](https://www.elastic.co/search-labs).

Please note this code is really _simple_. To make sure your tests are robust and efficient (especially if you have many of them!), please refer to two other blog posts: [Faster integration tests with real Elasticsearch](https://www.elastic.co/search-labs/blog/elasticsearch-integration-tests-faster) and [Advanced integration tests with real Elasticsearch](https://www.elastic.co/search-labs/blog/elasticsearch-improve-performance-integration-tests).

May the green tests be with you!
